An Andromeda API is a pure-PHP transactional REST-ish API.

public mixed ROOT = __DIR__ . '/'

Functionality is divided between the main framework core and apps that implement the desired domain-specific functions. Requests are given as an app name, action, and optional parameters.

The entire lifetime of the request happens under a single transaction. Any exceptions encountered will roll back the entire request safely. Multiple commands can be given to run in a single request/transaction.
