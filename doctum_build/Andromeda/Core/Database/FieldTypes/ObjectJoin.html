ObjectJoin

class ObjectJoin extends ObjectRefs

A field that represents a many-to-many relationship with another object.

Like ObjectRefs this field type is mainly syntactic sugar but has the added benefit of automatically managing the sub-objects that join the two classes together. Example - joining together Accounts and Groups requires a GroupMembership join object. The class is basically an ObjectRefs to a collection of join objects, but that uses an SQL JOIN query to load the actual joined class in LoadObjects()
