ObjectRefs

class ObjectRefs extends Counter

Represents a collection of (non-polymorphic) objects that reference this one.

Essentially is a syntactic-sugar/caching field that allows loading the array of objects without having to call into their class to load ones that reference us. The practical value is that the field stores a reference counter and acts as a cache.
